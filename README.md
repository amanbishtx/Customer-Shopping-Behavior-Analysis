# Customer Shopping Behavior Analysis
**End-to-End Data Analytics Project (Python | SQL | Power BI)**

## Business Problem
A retail company wants to better understand customer shopping behavior to improve sales, customer engagement, and marketing strategies.  
This project analyzes customer transaction data to identify purchasing patterns, customer segments, and key factors influencing buying decisions.

---

## Dataset
- **Total Records:** 3,900 transactions  
- **Total Columns:** 18 features  

The dataset includes:
- Customer demographics (Age, Gender, Location)
- Purchase details (Product, Category, Purchase Amount)
- Shopping behavior (Discount, Shipping Type, Subscription Status)
- Customer activity (Previous Purchases, Frequency, Review Rating)

---

## Project Workflow

### 1. Data Cleaning & Preparation (Python)
- Handled missing values in the dataset  
- Standardized column names  
- Created new features such as **Age Groups** and **Purchase Frequency**

### 2. SQL Business Analysis (MySQL)
Performed SQL queries to answer key business questions:
- Revenue generated by gender
- Top-rated products
- Customer segmentation (New, Returning, Loyal)
- Impact of discounts on purchases
- Comparison of spending between subscribers and non-subscribers

### 3. Data Visualization (Power BI)
Built an **interactive Power BI dashboard** to visualize:
- Sales by product category  
- Revenue by age group  
- Customer segments  
- Subscription vs non-subscription spending patterns

## Key Insights
- Loyal customers contribute significantly to overall revenue.
- Some products depend heavily on discounts to drive sales.
- Subscribers tend to spend more on average compared to non-subscribers.
- Certain age groups generate higher revenue for the business.
